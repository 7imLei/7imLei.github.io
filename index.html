迪拜自由行

发表于 2019-06-11 | 分类于 Tim

写在前面的话

有很多比沙漠更恐怖的事情，比如分离，比如背叛；
有很多比跳伞跟危险是存在，比如人性，比如私心。

目前我们这个阶级应该是没有多少时间筹备长期的旅行计划，旅行团什么的我就没有必要说了，完全是两码事，团游了很多，自驾也游了一些小地方，但是DUDU这次自驾刷新了我所有的体验，最多利用假期提前两周就飞了，因此我在下面说的每一个字都是很值得参考的，小到当地风情的融入小费应该怎么支配，大到行程机票和交通方式怎么选，我会根据我的行程一一回忆并陆续补充，另外，我的网站空间有限，尽量用文字，图片会很少，后期图片我单独换个图床给大家欣赏，那么开始吧。

定制行程安排

DUDU最忙的旅游季节中国的寒假和长假期间，那个时候机票的价格，酒店的价格很贵，人也很多，体验可能会有折扣，然后我们大胆的选择斋月时间正好也是高考ing的时间，两个加在一起，机酒就很便宜了，到底多便宜每年都不一样，所以我说出来也没有任何意义。游玩行程可以到了在提前一天去定，建议在酒店订，携程订的都是和中国人一起玩，遇到不清白的，乱带节奏的烦死你。

斋月：在一个月的时间里面白天不能在户外吃喝任何东西，除了旅游景区，很多地方都是关门的，场景和中国的春节大街上的场景一样，但是大型商场的美食城一样的地方和一些楼层的餐厅是用围挡挡起来，黑布遮盖起来，冰激凌和饮品店，也是只能打包到饮食区域吃，总体来说，并没有很大的影响，行程里面没有安排去大街小巷转悠所以当地的苍蝇馆子我们没有体验。

机票&飞行

直飞永远都是首选，如果能坐上阿联酋的航空那真是极好的，我们的直航是南航的，机型是波音787-8整体位子比较紧凑，前排第一排的位置是要加钱才能选的，600-300这些位子到了起飞前两天就不用加钱了，因为真的没有人去选哈哈，我个人建议选飞机尾巴上的2个位置，经济舱一排都是3人，只有飞机尾巴最后的2排是2人位的出来走动和旁边的空位放行李袋都是非常方便的，一开始可以把最后2排都占着，最后一排基本是没有人的旅行团的团票也是先锁中间的座位，那就可以睡在2个位置上，真是不错的选择。一共飞行8个小时，一直顺风的话会快40分钟到达，飞行路线是到云南再到东南亚，在伊朗的下方海上绕了一下路再去阿联酋，航线比去西班牙经过阿富汗还安全，过了云南不久会经过并看到喜马拉雅山，8800米的山10000飞行高度，夕阳下的喜马拉雅山真的非常美。

酒店

重要的事情我只说一遍，去酒店官网上定房间不要怕麻烦，在携程订价格一样没差多少，主要是你的积分只能给携程，不能划到你名下，特别是我们喜欢住一个品牌酒店的人。酒店有很多级别，高区低区、看海不看海、带早不带早、大床、双床、行政房行政房空间稍微大一点有桌子和更好的沙发其实没必要 我们没有选择棕榈岛上的酒店因为出行可能没有市区方便，我在卫星地图上面也看了，房间大部分都是朝着内陆看岛的，那种面海的即时在淡季价格也很高，而且可能订不上，并且人工岛，沙滩是对内的人工沙滩，不是对海的，也没什么意思啦，因此综合考虑，选择一个市区的高楼层酒店交通方便才是最重要的，较紧凑的行程只是睡个觉，如果是去轻松的休闲的玩法可以考虑沙屋海屋的别墅酒店，在DUDU真没必要。

出行交通

这里只有一条轻轨，出门出租车打表据说还是很贵，不如租车，我是尝到了租车的甜头，流程并没有那么复杂，下了飞机就是柜台，办了手续就去停车场取车，还车也在停车场。
DUDU附近开车必须去的有2个地方，一个是阿布扎比、一个是阿莱茵，这两个地方开车都需要一个半小时的车程，如果打车就不现实了，轻轨也不到，如果租车，去几次都没有问题，只要有人换着开车那是更好的出行选择。

游玩的项目

我按照从近到远的顺序来写
1.世界最大的商场，迪拜商场和迪拜塔可以玩一天，附带商场里面的水族馆都要几个小时玩，到了晚上还有半个一场的音乐喷泉看。
2.老城区的街道和以旧修旧的商业区和咱们武汉天地有一点像，老城区逛完了就花一块钱坐木划子10分钟过河去黄金市场和香料市场，有点像我们的汉正街小商品市场。
3.朱美娜海滩，那边在有工地，所以挡了一些沙滩与帆船酒店的景。
4.棕榈岛浏览和上面的大型水上世界。
5.跳伞等海上飞机项目就在棕榈岛旁边不远。
6.乐高主题公园和卡车主题餐厅都在去阿布扎比的高速路两边。
7.阿布扎比的老博物馆。
8.阿布扎比的白色清真寺、法拉利主题公园、华纳主题公园。
9.阿布扎比的海上卢浮宫，这个要看好需要一整天。
10.阿莱茵的绿洲、老博物馆目前关闭维修开业未知、骆驼交易市场。
11.阿莱茵Wadi Adventure水上世界、Jabal Hafeet山顶俯瞰阿曼和阿莱茵。
12.冲沙，地方在阿莱茵与迪拜城区之间的一个地方，各种营地，根据你报的团来调整的你也不知道具体营地是哪个。

以上都是重点必玩项目，先去哪个后去哪个这个可以根据实际情况来定，当然还有很多地方可以去，这里遍地都是家乐福，夜光花园、还有鹦鹉公园等等。。。可以在猫途鹰这个APP里面去找。

我用一张图来说，这些绿色的小旗子都是我标记要去最后还是没有去成放弃了地方。所以你说迪拜几天就没有什么玩的了我告诉你那是你不会玩，呆在国内算了。

1Day

下午的飞机，4个小时时差，飞了8个小时，看了3个小时的日落，南航的飞机餐难吃的想哭，因为落地要开车，飞机上助眠的酒也不敢喝，当然我觉得也罢，法航的酒都一般般，你想在南航上喝到什么？进了飞机直奔最后一排，2个位置旁边空了很多，放下手提袋，要什么直接伸手去拿非常方便，面前的桌板除了自己的，前排的多的桌板也可以用，有2个桌板放吃吃喝喝，一个大的无人空间放杂物，如果后排没人，2个位置还能睡一下，这个不比商务舱差呀。
落了以后坐一小段摆渡火车就到入关地方了，拿着护照就过海关了，新一点的电子护照就不用拍对了，直接去刷一下护照，照一下瞳孔就算进DUDU了哇。拿了行李没走多远就到了租车的门店，提交了预约表，核对了国际驾照和中国驾照，刷了一个押金，就发了一个取车单，让我们去制定停车场，一开始我们走错了，停车场真是太大了，问了一个保安，就取车了，到了取车点的办公室这里有很多的租车公司都在这里有柜台，突然问了我一句你多大了？我说33了，柜台的表情感觉我不像？我还很年轻么哈哈，一会儿分配了一个取车小哥哥就带我们去看车了。之前我们定的是路虎5座，路虎7座每天贵200呢买不起，可是单子上却把路虎划掉了，改写道奇，我很不爽，看到车以后一定要吵架，都背好发脾气的英语句子了。看到车以后我就傻逼了，5座路虎是没有，给我们了一个几乎全新的道奇7座，而且道奇是我最喜欢的品牌，可怕的是这一款在国内是没有的，进口都需要60多万，当时就给了消费给小哥哥心里觉得挺难过的，为什么中国人总在算计别人，麻将打多了？小哥哥在手机上圈画了几个微小的伤我签了字就开走了。我找不到大灯！！！！一路开一路早车灯，黑灯瞎火开了一会，突然回忆以前开大众系列的车也有找不到大灯经历，难道和大众一样？果然！一毛一样。在面板左下边单独的大灯开关没有GPS支架，同行小伙伴在副驾驶举着手机导航到酒店的。到了酒店门口，一群人就围上来了，给你开车门的，给你拿东西的，给你卸行李的，车钥匙给泊车小哥哥，什么都不操心的拿着随身包就去前台办入住了，这种体验，怎么描述呢，我这还是一个穷人初来玩，有钱人的生活我这样的土逼真的想都不敢想啊。到了房间行李就到了，给了小费小哥哥简单给我们介绍了一下房间和电器使用就走了。尼玛，落地窗脸上就是迪拜塔，2米大床，浴缸的浴缸，淋浴的淋浴，马桶也是一个单间，沙发写字桌，水吧有illy的胶囊咖啡机和咖啡，各种茶小枣子甜品全是免费的，我就想还有什么理由不选择这个酒店呢？这个价格在香港丽晶后面还有块墓地！本来要去酒店酒吧喝一点的，可是太累了就睡觉了。

2Day

我从来都不倒时差的，是不是年纪大了，突然就醒了，一看时间早上5点多，国内上午10点，是我自然醒最晚的时间。老婆一直都很能睡觉哇，还在睡，起来以后就开始闲逛拍延时摄影什么的，准备今天的出行计划，在网上查看了一些攻略注意事项什么的。因为斋月的原因怕是有影响我们去了迪拜商场停车准备去迪拜塔，看到一个售票台，一张不卖，要买就要和水族馆的票一起买，我们自己在网上尝试只买迪拜塔的票失败了就买了联票。可能是因为淡季原因，10分钟的队伍都没有排到就上去了，电梯一秒2.5层的样子，这个速度还是比我家的电梯快一点，我们家电梯号称城市最快电梯，一秒2层的速度，电梯里有投影动画告诉你现在已经超过什么什么建筑了，不久就到125楼了，明显有气压变化。楼上就是拍照尿尿转圈圈，纪念品。出了塔就到商场里面看水族馆，一开始以为就是一个几层楼的大水族馆，没想到逛完了水下长廊，楼上还有一系列的动物可以看，小到虫子蜘蛛水母，中到鹦鹉企鹅猫头鹰，大到鲨鱼鳄鱼，真是可以啊。

3Day

酒店早餐还是丰盛的，下的面条，炒的饭，各种面包西餐沙拉，五花八门老干妈。不敢吃很多，中午到了跳伞飞机场开始签生死状买保险，排队等教练领人，可以在屏幕上看自己的飞机和组员。一个教练两个摄像师为一组，一个飞机5组15个人一口气带走，教练帮你捆好跳伞背带以后教你出仓的动作，出仓后的动作，落地前的准备动作，强调你一定要看摄像师什么的注意教练的口令和手势等，交代完了，就开始拍跳伞前的VCR了。
飞机上的位置很小，感觉没有多余的空间，螺旋翼飞机噪音很大，一路上教练都在和我们开玩笑似乎是在让我们不要紧张，摄像师莫妮卡也一直在给我拍照和我讲笑话，捕捉我开心的镜头，不久温度就变化了，到了4000米以后感觉就有点冷了，还没搞明白要干什么，教练就把我往他大腿上放，感觉要做羞羞的事情，这个时候一个连体滑翔服的摄像师也一起起哄坐到我的腿上。玩起了3P，虽然我一点都不紧张，感觉跳伞也没有什么可怕的，但是在这样和不熟的人玩到了一起，也很畅快呀。飞机跳伞的舱门绿灯亮了，一组人已经下去了，教练绑好了我的安全扣又检查了一遍就开始一屁股一屁股的外外面挪动，挪到了舱门看到已经下去的伙伴也没什么担心，摄像师抓着舱门口的扶手，半个身体悬挂在外面拍摄我跳出的动作，此刻我还没有来得及摆出一个完美的表情，三个人就一同一二三出去了，3秒钟有过山车的失重感，一会就没有了，风打脸其实就和没戴头盔骑摩托到80码左右那种感觉一样呼吸的空气很冰凉有一点游泳呛水的感觉，因为你的呼吸道进入了大量的空气，如果姿势正确身体感觉浮在了天上和鸟一样飞翔很开心享受就好，这个时候头的方向就很重要了，另外你又要找镜头，所以一些列要做的事情有点多，我们没有真的详细去查询应该怎么做才能拍摄的最好的攻略，所以2分钟不到的自由落体的时间一会儿就过去了。这里需要强调，腿和身体一定要跳出后保持线上弧形香蕉的样子，手臂放松就可以了，不用用力伸直，和教练一起香蕉姿势是防止在空中打滚或者难以控制平衡浪费自由落体拍照的时间，当然教练会放出一个小平衡伞帮助我们不会过多纵向翻滚失去平衡，这一些列拍照的时间真的很短，教练就开伞了，开伞一瞬间有一股很大向上拉扯，这是第一步开伞为了初步减速并能够让伞保持正确展开位子，马上第二个拉扯就来了第二步开伞是把整个伞完全打开，完全打开的伞就可以开始滑翔并控制方向了，如果教练让伞加速下降还是会有失重感，左边转转右边转转，看到眼前很美棕榈岛和DUDU各种奇形怪状的高楼大厦，我还和教练开玩笑，我的酒店不远就在那里，我们直接落那边好吧？他还真去转了一下啊！吓死宝宝我了。落地以后又拍摄了一个VCR就结束了，等40分钟拿他们剪辑我刚刚拍的照片与视频的U盘就可以走了。一定要吃晕车药 某人在滑翔的时候特别难受，左右转圈导致恶心。最后下来吐了3次平静很久。
因为知道跳完会不舒服，所以中午在附近的家乐福逛了一下买了一个中国产夹着手机的用于导航夹子免得老要人举着导航哭死，中午吃了一点特别的快餐，就去老城区了。

我和咖啡

发表于 2018-10-17 | 分类于 Tim

速溶咖啡

小朋友们在亲戚家过春节，转钟之后客厅里的难忘今宵播放完，客厅里面就是麻将声了，小孩子们在一个卧室倒在一起睡着。电视里面一直在播放《猫和老鼠》VCD动画片，慢慢开始的不是困意，是饿了，小朋友过春节吃年饭不就是挑几个爱吃的菜，大口喝饮料，最后被大人硬塞两个饺子包在嘴里就跑了节奏，我相信大多数在城里春节的童年都是这样难得如此放肆一回。

身边的玩伴都饿着睡着了，偶然在柜子上看到一袋雀巢速溶，在没有经大人批准的情况下，就这样生吃了。入口很苦，突然发现原来苦后是甜味，第一次学会品尝的滋味应该就是这一次吧。任何时候，小孩子对食物的感知就是入口第一感觉，而苦中带甜的味道让我发现原来世界上还有这样的东西。

接着有意思的事情发生了，能够感觉自己的小心脏要蹦跶出来一样，慢慢开始紧张，似乎可以回忆起眼前《猫和老鼠》中每一集每一个即将发生的细节，再然后就是疲惫无比瞌睡，但就是合不上眼睛睡不着，到了后半夜，一个人躺着发呆，想睡觉又睡不着。

精品咖啡

接触到自给咖啡以后我变化了很多，什么才是超于传统咖啡、商业咖啡、逼格咖啡的新领域，是一种在有限条件下尽量全力的精神，是一种克服重重阻碍，标新立异又不忘初衷的态度，也是一种本性归宿。

多少海拔的生豆与性价比最高的渠道已经是要考虑的重点问题，然而网红的打卡这种营销战略已早被诗阳定义成贬义，在我这里更多的是一杯狗屎一样的咖啡创意咖啡还鼓舞大家为其点赞的手段，正在腐蚀并迅速蔓延整个圈子，因此不要觉得自己有多厉害，当你布粉的一瞬间，当你开气阀的那一瞬间，就明朗起来，后面的时间是消费，还是被试验。经过学习，我已经刻苦萃取了一年的咖啡，打了一年的奶沫了，我依然拉出狗屎一样的拉花，当朋友喝到第一口的时候发现有点内容的时候我就安心了，是的，这种自给咖啡精神一直在影响我，做好一件简单的事情没有那么随意满足，降低标准，没有不好用的设备和器具，只有浮躁自作聪明的人。

装修房子这一篇就够了

发表于 2018-10-14 | 分类于 Tim

前言

不尽人意

  • 玄关到户外没有留一根网线，监控使用。
  • 阳台的窗子没有外开。
  • 所有浴室柜的柜子里没有留足够的插座，电动牙刷什么的放在外面充电很丑。
  • 床头柜旁边的插座太高了，没有必要做那么高也很难看。
  • 阳台和厨房有地漏的地方斜度不合理，积水不容易排出。
  • 地板瑕疵后期换板子是切割再拼装，不如之前牢固，能不换就不换的。
  • 旅行箱太多，没有合适放大旅行箱的地方。
  • 电视柜后面的埋管应该在接近地面一些，一定要低于电视柜表面。
  • 水泵预留的位置忘记考虑了，进水地方安装水泵非常勉强。
  • 烘干机的预留位置考虑不够周密，洗衣机和烘干机最好上下放置。
  • 暗卫不要放任何毛巾挂物，阴干的毛巾很恶心。
  • 负压情况，新风环境窗子是密封的，所以排风扇开起来负压导致下水管翻臭味。
  • 开关的双开与单开，设计不够精细，有很多地方没有必要双开，单开就够了。
  • 家中的HDMI2.0是15米的预埋，超过8米就应该用光纤HDMI的，部分设备4K无法播放。

像狗一样躺下来

发表于 2018-07-13 | 分类于 Tim

慵懒倒下

真的，有时候我就原地一趟给这个世界来一次不要脸的碰瓷。永无止境的负能量实在没有力气再抵消了。
每一天的清晨，小区小道上的狗屎，巷口发酵的恶臭，不安规矩靠右的人群，抢着位子堵你前面的傻子，按着不放的喇叭声，追逐风的电瓶车，我蠢逼一样呆呆站在斑马线上等红灯，似乎脑海里永远都在回响红色旗帜，每天都一样的口号，我不禁一笑。
当我疲惫于此的时候，来到一个心灵深处的港湾，找一个凉快的小地盘就此躺下，叹口气，看着世间万物匆匆流过，外面发生的事情就让他们去，我没有义务牺牲自己来优化他们。

见越多，烦就越多

人性的主流就是跟着大家的烦恼过着一日一月，永远都和差的比较看谁吃的狗屎更臭，当麻痹自己的同时，你国真正的释怀了。阿尔卑斯山脉的融雪正在苏醒小骚牛和小骚花草；思连伊森林小鹿追逐打闹；圣家堂的塔尖又封顶了一个；格雷梅国家公园里澎湃的火焰点燃了热气，一群球正在缓缓上升；伊斯坦布尔的浓稠咖啡随着祷告喇叭声细微回荡；仿佛一瞬间，苏梅岛的海滩上浩荡的渔船满载而归。
剩下的就都是现实了。

和物品打交道

工作累了，有人喜欢在家做做饭给自己吃。吵架吵累了，有人开始不停的做家务、不停的吃东西、不停的买包包。似乎做这样的事情会比沟通、解释来得容易简单。
